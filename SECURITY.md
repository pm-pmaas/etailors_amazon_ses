Security Policy

Thank you for helping keep this project and its users safe.

Reporting a vulnerability
- If you discover a security issue or vulnerability, please report it privately by emailing: info@e-tailors.com
- Include as much detail as possible: a description of the issue, steps to reproduce, potential impact, and any suggested remediation.
- Please do not open public issues for security reports.

Disclosure and response
- We will acknowledge receipt of your report as soon as possible and aim to provide an initial response within a few business days.
- Once the issue is validated, we will work to address it and coordinate a responsible disclosure timeline with you.

Thank you for your responsible disclosure and for improving the security of this project.
